TabsLayout 选项卡

让用户可以在不同子任务、视图、模式之间切换，它具有全局导航的作用，是全局功能的主要展示和切换区域。

何时使用​

  • 当需要对页面内容进行分类或区隔时使用。
  • 当需要对页面控件进行压缩，提升页面空间利用率时。
